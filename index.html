Technologies used in the project

PostgreSQL

PostgreSQL is a powerful, open source object-relational database system with over 30 years of active development that has earned it a strong reputation for reliability, feature robustness, and performance.

Ruby on Rails

The framework, written in the Ruby programming language, implements the Model-View-Controller architectural template for web applications and integrates them with the web server and the database server. Is open source software and is distributed under the MIT license. Created by David Heinemeyer Hanson based on his work at 37signals on the Basecamp project management tool and released in July 2004. 3, which combines the best features of both frameworks.

Docker

Tools for managing isolated Linux containers. Docker complements the LXC toolkit with a higher-level API that allows you to manage containers at the isolation level of individual processes. In particular, Docker allows you to run the arbitrary processes in isolation mode without worrying about the contents of the container and then transfer and clone the containers formed for these processes to other servers, taking on all the work of creating, maintaining and maintaining containers.

Amazon S3 Bucket

Data storage service, one of Amazon Web Services. The service provides the ability to store and receive any amount of data, at any time from anywhere on the network, ie the so-called file hosting. With the help of Amazon S3 high scalability, reliability, high speed, inexpensive storage infrastructure is achieved. It first appeared in March 2006 in the United States and in November 2007 in Europe.
